EXAMPLE EXECUTION FLOW
======================

User runs: python main.py

1. Logging configured (INFO level)
2. WeatherAgent initialized:
   - GOOGLE_API_KEY loaded from .env
   - genai.Client created
   - Tools loaded: get_current_weather, get_weather_forecast
   - Conversation history initialized

3. Example queries executed:
   - Agent.chat("What's the weather in San Francisco?")
   - Gemini receives query + available tools
   - Gemini calls get_current_weather("San Francisco")
   - Tool returns JSON weather data
   - Gemini generates human-readable response
   - Response printed with tool names

4. Interactive mode started:
   - User enters weather questions
   - Agent processes same way as examples
   - Loop until 'quit' or KeyboardInterrupt


TOOL CALLING MECHANISM
======================

1. User asks: "What's the weather in Paris?"

2. Agent calls Gemini API with:
   - User message
   - Available tools (get_current_weather, get_weather_forecast)
   - System instruction (weather expert role)

3. Gemini analyzes query and decides:
   - Tool needed: get_current_weather
   - Parameters: location="Paris"

4. Agent executes tool:
   - Calls get_current_weather("Paris")
   - Gets JSON: {temperature: 72, humidity: 65, ...}

5. Agent sends results back to Gemini:
   - "Here's the tool response..."
   - Gemini formulates final answer

6. Response shown to user:
   "The weather in Paris is currently 72°F and partly cloudy..."


INTEGRATION POINTS
==================

To add real weather API (OpenWeatherMap):

1. Get API key from openweathermap.org
2. Add to .env: WEATHER_API_KEY=...
3. Update get_current_weather() in src/tools/core_tools.py:

   import requests
   response = requests.get(
       'https://api.openweathermap.org/data/2.5/weather',
       params={'q': location, 'appid': WEATHER_API_KEY}
   )
   data = response.json()
   return json.dumps({
       'temperature': data['main']['temp'],
       'humidity': data['main']['humidity'],
       ...
   })

4. Same for get_weather_forecast() using forecast endpoint
