PROJECT OVERVIEW
================================================================================

The Weather Agent is a production-ready Python application that leverages
Google's Gemini 2.0 API to intelligently answer questions about weather.
The agent uses function calling to automatically invoke weather tools based
on user queries, enabling natural language weather information retrieval.

KEY FEATURES:
- Natural language weather queries using Gemini 2.0
- Automatic function calling for current weather and forecasts
- Conversation history tracking for context-aware responses
- Type-safe data models using Pydantic
- Clean, modular architecture ready for extension
- Comprehensive error handling and logging

================================================================================
KEY GENERATED CODE
================================================================================

1. MAIN ENTRY POINT (main.py)
   - Initializes WeatherAgent with logging
   - Runs 4 example weather queries
   - Enters interactive mode for user input
   - Proper error handling and graceful shutdown

2. BASE AGENT CLASS (src/agents/base_agent.py)
   - WeatherAgent class with Gemini 2.0 integration
   - Automatic tool loading via _setup_tools()
   - Natural language query processing via chat()
   - Conversation history tracking
   - Tool call execution and result processing
   - System instructions for weather expertise

3. CORE TOOLS (src/tools/core_tools.py)
   - get_current_weather(location) - Returns current conditions
   - get_weather_forecast(location, days) - Multi-day forecasts
   - get_weather_alerts(location) - Active weather warnings
   - format_weather_response(json) - Human-readable formatting

4. CONFIGURATION (src/config.py)
   - Loads GOOGLE_API_KEY from .env
   - Initializes Gemini client
   - Error handling for missing credentials

5. TYPE SAFETY (src/schemas/types.py)
   - WeatherCondition - Current weather model
   - ForecastDay & WeatherForecast - Forecast models
   - WeatherAlert & WeatherAlertsResponse - Alert models
