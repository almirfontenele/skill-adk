HOW TO GET STARTED
================================================================================

1. Navigate to the project directory:
   cd /home/dgs-admin/projetos/skill-adk/python-adk-creator-workspace/iteration-1/eval-3-research-assistant/with_skill/

2. Create virtual environment:
   python -m venv venv
   source venv/bin/activate

3. Install dependencies:
   pip install -r requirements.txt

4. Configure API key:
   cp .env.example .env
   # Edit .env and add your Google API key

5. Get your API key from:
   https://aistudio.google.com/app/apikey

6. Run the application:
   python main.py

7. Try a query:
   You: What are the latest AI trends?
   [Agent searches web and provides response]

8. Exit:
   Type 'quit' or 'exit'
