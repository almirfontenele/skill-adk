HOW IT WORKS
================================================================================

FUNCTION CALLING FLOW:

1. User Provides Input
   "What is 15 plus 8?"
   
2. Agent Receives & Processes
   - CalculatorAgent.interact(user_input)
   - Creates tool schema in JSON format
   - Sends to Gemini with available tools
   
3. Gemini Analysis & Decision
   - Analyzes the request
   - Determines which tool to call
   - Requests: add(a=15, b=8)
   
4. Tool Execution
   - Dispatcher calls execute_tool("add", {"a": 15, "b": 8})
   - Function executes: 15 + 8 = 23.0
   - Result logged and returned
   
5. Response Formatting
   - Tool result incorporated into response
   - Formatted for user consumption
   - Logged for audit trail
   
6. User Receives Answer
   "I performed the following calculation(s):
    1. Add operation result: 23.0"

EXTENSIBILITY EXAMPLES
================================================================================

Adding a new tool is straightforward:

1. Define function in src/tools/calculator.py
   def power(base: float, exp: float) -> float: ...

2. Add JSON schema to get_calculator_tools()
   { "type": "function", "name": "power", ... }

3. Update execute_tool() dispatcher
   elif tool_name == "power": return power(...)

4. Update exports in src/tools/__init__.py

The tool is immediately available to Gemini!
